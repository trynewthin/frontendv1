### 关键修改点：

1. **移除了内容容器的宽度限制和外边距**：
   ```css
   .detail-container {
     max-width: 100%;
     margin: 0;
     width: 100%;
   }
   ```
   这样内容容器会占据整个页面宽度，没有任何边距限制。

2. **移除了圆角边框**：
   ```css
   /* 移除了 border-radius: 8px; */
   ```
   这样容器会完全贴合页面边缘，没有圆角。

3. **为内容区域设置了最大宽度和居中对齐**：
   ```css
   .car-info {
     max-width: 1400px;
     margin-left: auto;
     margin-right: auto;
     width: 100%;
   }
   
   .header-section {
     max-width: 1400px;
     margin-left: auto;
     margin-right: auto;
     width: 100%;
   }
   ```
   这样内容会在超宽屏幕上有一个合理的最大宽度，但仍然会占据整个可用空间。

4. **简化了移动端的样式**：
   ```css
   @media (max-width: 768px) {
     .detail-container {
       padding: 1rem;
     }
   }
   ```
   在移动设备上使用更小的内边距，但不再有外边距限制。
